# redis安装

# Linux

  1. 安装包地址

    官网下载地址 (opens new window)

  2. 解压缩

    tar -zxvf redis-6.0.8.tar
    1

  3. 编译测试

    sudo make test
    1

  4. 安装

    sudo make install
    1

  5. 设置后台启动

    编辑 redis.conf，修改如下内容。

    #配置后台运行
daemonize yes
    1
    2

  6. 启动

    sudo redis-server redis.conf
    1

  7. 进入交互命令行

    reids-cli
    1

# 虚拟机

  1. 复制安装包到 /usr/local

  2. 解压安装包

    tar -zxvf redis-3.0.4.tar.gz
    1

  3. 执行make

  4. 执行make install

  5. 修改配置文件 redis.conf，将 daemonize no 改为 yes ，让 redis 服务在后台启动。

  6. 将 redis.conf 复制到 /usr/local/bin

  7. 启动 redis

    redis-server redis.conf
    1

  8. 测试连接

    redis-cli -h 192.168.126.110 -p 6379
    1

  9. 防火墙开放6379端口

    firewall-cmd --zone=public --add-port=6379/tcp --permanent
firewall-cmd --reload
    1
    2

  10. 使用 RedisDesktopManager 进行外网连接测试
