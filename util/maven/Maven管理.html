# 上传包到私服

# 概述

在公司工作的时候，使用的 Maven 仓库是公司私服。有的时候对接第三方 SDK，需要把第三方的 SDK 手动上传到公司私服，这里记录一下上传步骤。

https://www.cnblogs.com/htyj/p/8024791.html (opens new window)

	<gruopId>im.youdu</gruopId>

</dependency>
