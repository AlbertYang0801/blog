# vuepress总结

# 一、环境安装

# 1.使用brew安装node环境

  • 安装 brew

    /bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)
    1

  • 安装 node

      brew install node
    1

  • 查看版本号

    node -v
    1

# 2. vuepress

前提：安装node环境

  • npm 安装国内镜像

    npm config set registry=http://registry.npm.taobao.org -g
    1

  • 使用 npm 安装

    npm install -g vuepress
    1

  • 查看版本号

    vuepress -version
    1
