# 同步代码块和同步方法

# 同步代码块

重点是保证多个线程的锁对象是一致的。

  • this作为锁对象
//this的锁对象指当前类的实例
synchronized (this) {
    i++;
}

# 同步方法

synchronized 加在普通方法上或者静态方法上，可实现同步方法。

  • 同步普通方法 普通同步方法，锁对象为当前类的实例对象等同于this。
    public synchronized void increaseI() {
        i++;
    }
    等价于
    public void increase() {
        synchronized (this) {
            i++;
        }
    }
