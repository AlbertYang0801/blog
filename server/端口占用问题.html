    # 端口占用问题

    初学tomcat的时候经常会遇到端口占用问题，还有项目未成功关闭时也可能会遇到端口占用问题。以下是本人常用的两种解决端口占用问题的方法。

    # 1.查看端口号

    #查看所有端口号 netstat -ano
#查看端口号 netstat -ano|findstr 8005 
#结束端口号进程 taskkill /pid 27456 /f

    # 2.结束指定端口号进程(mac)

    sudo lsof -i :8100（查看指定端口号)
sudo kill -9 8100 （结束端口号进程）
